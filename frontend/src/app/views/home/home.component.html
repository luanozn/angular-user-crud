User CRUD Homepage Informações Projeto desenvolvido utilizando o framework Angular Projeto de Criação, Leitura, Atualização e Deleção (Create, Read, Update, Delete) de usuários simples, utilizando o framework Angular, o banco de dados MongoDB, e uma API de autoria própria que utiliza do Framework Spring Boot, juntamente com a linguagem Java para enviar emails aos usuários que realizam interações com a aplicação. O back-end do projeto (Banco de dados, e API de envio de emails) foi hospedado na plataforma Railway, no link a seguir: [ https://spring-mail-sender-production.up.railway.app ], onde as requisições pro mesmo estão disponíveis no caminho /users. Já o front-end do projeto se localiza no Netlify, que é outra plataforma de hospedagem.
